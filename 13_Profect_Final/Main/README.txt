Описание проекта

Анализ клиентов регионального банка и сегментация пользователей по количеству потребляемых продуктов.
- Иисследовательский анализ данных,
- Сегментация пользователей на основе данных о количестве потребляемых продуктов,
- Формулировка и проверка статистические гипотезы.
    
Подготовка презентации по итогам исследования.
Подготовка дашборда:
1. Диаграмма, отражающая количество клиентов в зависимости от числа используемых продуктов.
2. Индикатор процента ушедших клиентов.
3. Фильтр дашборда по городу.
